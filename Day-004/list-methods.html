Python List Methods Guide

Python lists are fundamental data structures. Let's explore their main methods with clear code examples and their results.

Adding and Inserting Elements

1. append()

Adds a single element to the end of the list.

fruits = ['apple', 'banana', 'cherry']
fruits.append('strawberry')
print(fruits)
Output: ['apple', 'banana', 'cherry', 'strawberry']

2. extend()

Extends the list by appending all the elements from an iterable.

list1 = [1, 2, 3]
list2 = [4, 5, 6]
list1.extend(list2)
print(list1)
Output: [1, 2, 3, 4, 5, 6]

3. insert()

Inserts an element at a specific position.

numbers = [10, 20, 40, 50]
numbers.insert(2, 30)
print(numbers)
Output: [10, 20, 30, 40, 50]

Removing Elements

4. remove()

Removes the first occurrence of a specified element.

letters = ['a', 'b', 'c', 'b', 'd']
letters.remove('b')
print(letters)
Output: ['a', 'c', 'b', 'd']

5. pop()

Removes and returns an element at a specific index.

items = ['pencil', 'notebook', 'pen', 'eraser']
removed_item = items.pop(1)
print(f"List after pop: {items}")
print(f"Item removed: {removed_item}")
Output:
List after pop: ['pencil', 'pen', 'eraser']
Item removed: notebook

6. clear()

Removes all elements from the list.

full_list = [1, 2, 3, 4, 5]
full_list.clear()
print(full_list)
Output: []

Searching and Counting Elements

7. index()

Returns the index of the first occurrence of an element.

colors = ['blue', 'red', 'green', 'yellow', 'red']
position_of_red = colors.index('red')
print(f"The first 'red' is at index: {position_of_red}")
Output: The first 'red' is at index: 1

8. count()

Returns the number of times a specific element appears in the list.

numbers = [1, 2, 3, 2, 4, 2, 5]
count_of_2 = numbers.count(2)
print(f"The number 2 appears {count_of_2} times.")
Output: The number 2 appears 3 times.

Reorganizing and Copying Lists

9. sort()

Sorts the elements of the list in place.

numbers = [3, 1, 4, 1, 5, 9, 2]
numbers.sort()
print(f"Ascending order: {numbers}")

numbers.sort(reverse=True)
print(f"Descending order: {numbers}")
Output:
Ascending order: [1, 1, 2, 3, 4, 5, 9]
Descending order: [9, 5, 4, 3, 2, 1, 1]

10. reverse()

Reverses the order of the elements in the list in place.

letters = ['a', 'b', 'c', 'd']
letters.reverse()
print(letters)
Output: ['d', 'c', 'b', 'a']

11. copy()

Returns a shallow copy of the list.

original_list = [1, 2, 3]
copied_list = original_list.copy()
copied_list.append(4)
print(f"Original list: {original_list}")
print(f"Copied list: {copied_list}")
Output:
Original list: [1, 2, 3]
Copied list: [1, 2, 3, 4]
Note: A simple assignment like copied_list = original_list creates a reference, not a copy. Changes to one list would affect the other.
