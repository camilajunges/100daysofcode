Python Built-in Functions

print()

Prints the specified message to the screen or to a file.

print("Hello, World!")

len()

Returns the number of items in an object.

my_list = [1, 2, 3]
length = len(my_list)
# length is 3

type()

Returns the type of the specified object.

my_var = "hello"
print(type(my_var))
#

range()

Generates a sequence of numbers, useful for loops.

for i in range(3):
    print(i)
# 0, 1, 2

input()

Reads a line from the user input.

name = input("Enter your name: ")
print("Hello, " + name)

sum()

Returns the sum of all items in an iterable.

numbers = [1, 2, 3, 4]
total = sum(numbers)
# total is 10

sorted()

Returns a new sorted list from the items in an iterable.

a = [3, 1, 2]
b = sorted(a)
# b is [1, 2, 3]

int() / str()

Converts values to integer or string types.

num = int("10")
text = str(100)

min()

Returns the item with the lowest value.

numbers = [5, 1, 9, 2]
lowest = min(numbers)
# lowest is 1

max()

Returns the item with the highest value.

numbers = [5, 1, 9, 2]
highest = max(numbers)
# highest is 9

abs()

Returns the absolute value of a number.

x = abs(-7.25)
# x is 7.25

round()

Rounds a number to a specified number of decimals.

x = round(5.7654, 2)
# x is 5.77
