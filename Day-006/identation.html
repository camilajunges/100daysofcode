Indentation in Python

In Python, indentation is more than just a style convention; it's a fundamental part of the language's syntax. It is used to define the structure of code blocks, replacing the need for curly braces or keywords found in other languages.

The Basic Concept

Indentation is what tells the Python interpreter that a line of code belongs to a specific block, such as an `if` statement, for loop, or a function definition. All statements within the same block must have the same level of indentation. 

# Example of an 'if' block
if 5 > 2:
    print("Five is greater than two!")
    print("This line is also part of the 'if' block.")
    # The two print statements are indented with the same number of spaces.

Standard Practice: 4 Spaces

The official Python style guide, PEP 8, recommends using 4 spaces for each level of indentation. This standard ensures code readability and consistency across different projects and development environments. While some editors may allow tabs, mixing tabs and spaces can lead to an IndentationError. 

# Example following the 4-space rule
def calculate_area(length, width):
    # This line has 4 spaces of indentation.
    area = length * width
    # This line also has 4 spaces.
    return area

Common Errors

One of the most frequent mistakes for beginners is the IndentationError. This happens when the indentation is inconsistent or incorrect, and it's a syntax error that will prevent your code from running. 

# This code will cause an IndentationError.
for i in range(2):
    print("Correct indentation.")
  print("Incorrect indentation.") # This line has a different number of spaces.

Nesting Code Blocks

Indentation is crucial for defining nested structures. Each nested block of code requires an additional level of indentation. This visual hierarchy makes the flow of control within your program clear and easy to follow. 

# Example of nested loops with correct indentation
for x in range(2):
    for y in range(2):
        print(f"({x}, {y})")
        # The print statement is indented two levels (8 spaces).
